## 조사 범위와 준수

- robots 확인: `https://www.baeksang.dev/robots.txt`
  - 일반 user-agent는 전체 허용이며 `/daily`, `/daily/archive`에 별도 Disallow가 없습니다.
  - SemrushBot/AhrefsBot 차단 규칙을 침범하지 않았습니다.
- 방문 페이지: 총 8개(상한 15 이하)
  1. `https://www.baeksang.dev/robots.txt`
  2. `https://www.baeksang.dev/daily`
  3. `https://www.baeksang.dev/daily/archive`
  4. `https://www.baeksang.dev/daily/2026-07-18`
  5. `https://www.baeksang.dev/daily/2026-07-17`
  6. `https://www.baeksang.dev/daily/2026-07-16`
  7. `https://www.baeksang.dev/daily/2026-07-15`
  8. `https://www.baeksang.dev/daily/2026-07-14`
- 분석 방식: 페이지 기능과 정보 구조만 분석했습니다. 픽셀 카피, CSS, 고유 selector는 재현하지 않습니다.
- 샘플 관찰: №084~088은 날짜별 15~21개 항목을 담고, 날짜 URL·아카이브·출처 집계·섹션별 묶음·내부 상세 스토리·외부 원문 링크를 반복적으로 사용합니다.

> 참고: 조사 페이지의 숫자 표시는 HN의 원 점수라기보다 출처 건수 집계 또는 편집 평점으로 관찰됐습니다. VoidNews가 HN 점수를 넣을 때는 Algolia `points`를 원 플랫폼 점수로 명확히 표기해야 합니다.

## 바로 이식 후보

- [ ] **날짜 고정 URL**
  - 무엇인지: 각 호를 `/daily/YYYY-MM-DD`처럼 고정된 날짜 경로로 보존하고 이전·다음·아카이브로 이동합니다.
  - VoidNews 이식 형태: Weekly/AB에 사람이 읽을 수 있는 날짜 alias를 추가하되 기존 canonical slug와 redirect 정책을 유지합니다.
  - 난이도: **S**
  - 기대 효용: 공유 링크가 최신 페이지 변경과 분리되어 과거 브리핑 재현성이 높아집니다.

- [ ] **호수 + 날짜 동시 표기**
  - 무엇인지: `№088 · 2026-07-18`처럼 연속 발행 번호와 실제 날짜를 함께 보여줍니다.
  - VoidNews 이식 형태: AB의 VOL 번호와 범위, Weekly의 주차와 발행일을 헤더 metadata로 함께 노출합니다.
  - 난이도: **S**
  - 기대 효용: 발표 회차와 실제 시간 범위를 한눈에 구분할 수 있습니다.

- [ ] **출처별 수집 건수 요약**
  - 무엇인지: HN·Reddit·GitHub·한국 매체 등 그날 포함된 출처 유형별 건수를 상단에서 보여줍니다.
  - VoidNews 이식 형태: Radar 보고서 부록에 `official/community/curator` 및 플랫폼별 seed·승격 건수를 자동 집계합니다.
  - 난이도: **S**
  - 기대 효용: 무엇을 얼마나 훑었는지 가시화되어 누락 의심을 빠르게 찾습니다.

- [ ] **섹션별 고정 질문**
  - 무엇인지: 새 소식, 직접 써볼 것, 반응, 배경지식처럼 독자의 목적에 따라 항목을 분리합니다.
  - VoidNews 이식 형태: `출시/업데이트`, `써볼 도구`, `커뮤니티 반응`, `연구·맥락`, `한국 관점`으로 분류하되 기존 ranking 데이터에서 결정론 매핑합니다.
  - 난이도: **M**
  - 기대 효용: 한 긴 순위표보다 독자가 원하는 행동으로 곧바로 이동할 수 있습니다.

- [ ] **상세 항목의 행동 필드**
  - 무엇인지: 핵심 설명 뒤에 독자가 얻는 것, 지금 할 일, 지금 중요한 이유를 반복합니다.
  - VoidNews 이식 형태: Top 발표 후보에만 `takeaway`, `tryNow`, `whyNow` 필드를 추가하고 일반 아카이브 카드는 짧게 유지합니다.
  - 난이도: **M**
  - 기대 효용: 뉴스 요약이 실제 수업·업무 행동으로 이어집니다.

- [ ] **외부 원문과 내부 해설 분리**
  - 무엇인지: 외부 원문 링크와 `/daily/stories/<slug>` 내부 상세 해설 링크를 별도 제공합니다.
  - VoidNews 이식 형태: 카드에는 `officialUrl`을 우선 표시하고, 내부 Deep Dive가 있을 때만 별도 `analysisUrl`을 제공합니다.
  - 난이도: **M**
  - 기대 효용: 독자는 원문 검증과 쉬운 해설 중 필요한 경로를 선택할 수 있습니다.

- [ ] **공식 1차 링크 우선 라벨**
  - 무엇인지: GitHub 저장소, 공식 제품·연구 페이지를 외부 링크로 직접 연결합니다.
  - VoidNews 이식 형태: 링크 라벨을 `공식`, `GitHub`, `연구`, `커뮤니티 반응`으로 구분하고 community seed는 공식 링크와 시각적으로 분리합니다.
  - 난이도: **S**
  - 기대 효용: discovery 경로를 출처로 오해하는 문제를 줄입니다.

- [ ] **짧은 항목과 깊은 항목의 밀도 차등**
  - 무엇인지: 대표 항목은 상세한 행동 필드를 주고 나머지는 제목·한 줄·원문 링크로 압축합니다.
  - VoidNews 이식 형태: Top 5는 확장 카드, 보강·Radar 항목은 compact row로 렌더링합니다.
  - 난이도: **M**
  - 기대 효용: 전체 커버리지를 유지하면서도 페이지가 과도하게 길어지는 것을 막습니다.

## 선택형 확장 후보

- [ ] **아카이브 검색**
  - 무엇인지: 제목·요약·얻는 내용 같은 텍스트를 검색합니다.
  - VoidNews 이식 형태: 정적 client-side 인덱스에 한국어/영문 title·summary·tags를 넣고 URL query로 검색 상태를 보존합니다.
  - 난이도: **M**
  - 기대 효용: 과거 모델·회사·도구 이력을 즉시 찾을 수 있습니다.

- [ ] **태그 필터**
  - 무엇인지: AI/LLM, Agent, Inference, Training 등의 주제 태그로 호를 좁힙니다.
  - VoidNews 이식 형태: 기존 post tags를 정규화하고 아카이브에서 다중 선택 필터를 제공합니다.
  - 난이도: **M**
  - 기대 효용: 발표 준비나 특정 제품 추적 시간이 줄어듭니다.

- [ ] **월별 달력 탐색**
  - 무엇인지: 달력에서 발행된 날짜를 보고 해당 날짜판으로 이동합니다.
  - VoidNews 이식 형태: Weekly/AB 발행일에 점 표시가 있는 월별 미니 캘린더를 아카이브에 둡니다.
  - 난이도: **M**
  - 기대 효용: 발행 공백과 cadence를 시각적으로 확인하기 쉽습니다.

- [ ] **발행 cadence 표시**
  - 무엇인지: 아카이브에 매일 08:00 KST 같은 갱신 주기를 명시합니다.
  - VoidNews 이식 형태: 격주 발행 예정일과 마지막 데이터 갱신 시각을 분리해 표시합니다.
  - 난이도: **S**
  - 기대 효용: 사용자가 새 호를 언제 기대할지 알 수 있고 운영 지연도 드러납니다.

- [ ] **HN 원 점수 병기**
  - 무엇인지: HN community 신호의 원 플랫폼 points를 제목 근처에 보입니다.
  - VoidNews 이식 형태: `community-hn` seed에서 `score`를 가져와 `HN 123 points`로 표기하고 VoidNews 편집 점수와 절대 혼합하지 않습니다.
  - 난이도: **S**
  - 기대 효용: 커뮤니티 화제성을 편집자 판단과 별도로 읽을 수 있습니다.

- [ ] **Reddit 커뮤니티·점수 병기**
  - 무엇인지: Reddit 항목에 subreddit과 원 score를 표시합니다.
  - VoidNews 이식 형태: `r/LocalLLaMA · 428`처럼 Radar 표면에만 표시하고 공식 카드의 신뢰 점수로 사용하지 않습니다.
  - 난이도: **S**
  - 기대 효용: 어떤 커뮤니티에서 반응했는지 맥락이 생깁니다.

- [ ] **항목별 출처 품질 라벨**
  - 무엇인지: 출처 또는 편집 평가를 항목 근처에 표시합니다.
  - VoidNews 이식 형태: 임의의 10점 평점 대신 현재 verifier 결과를 `공식 확인`, `교차 확인`, `검토 필요`처럼 근거 기반 상태로 노출합니다.
  - 난이도: **M**
  - 기대 효용: 숫자 권위 대신 검증 상태를 투명하게 전달합니다.

- [ ] **내부 Deep Dive 템플릿**
  - 무엇인지: 대표 뉴스마다 독립 상세 페이지를 두고 목록판보다 긴 맥락을 제공합니다.
  - VoidNews 이식 형태: 발표 메인 1~2개만 `/stories/<slug>` 상세 페이지로 만들고 source ledger를 하단에 붙입니다.
  - 난이도: **L**
  - 기대 효용: 브리핑 카드와 교육용 장문 해설을 분리할 수 있습니다.

- [ ] **전체 텍스트 복사**
  - 무엇인지: 날짜판 전체를 펼쳐 복사할 수 있습니다.
  - VoidNews 이식 형태: 접근 가능한 plain-text/Markdown export 버튼을 두되 내부 메모·risk flag를 제외한 공개 필드만 직렬화합니다.
  - 난이도: **M**
  - 기대 효용: 발표 대본, 메신저 공유, 개인 노트로 재사용하기 쉬워집니다.

- [ ] **메신저 공유 카드**
  - 무엇인지: 카카오톡 등으로 날짜판을 카드 형태로 공유합니다.
  - VoidNews 이식 형태: 우선 Web Share API와 링크 복사만 제공하고, 카카오 SDK는 키·개인정보·외부 의존성 검토 후 opt-in합니다.
  - 난이도: **M**
  - 기대 효용: 한국 사용자에게 브리핑 유통 경로가 짧아집니다.

- [ ] **RSS/Atom 피드**
  - 무엇인지: 조사 페이지 본문에서는 명시적 RSS 링크가 관찰되지 않았지만 일일/격주 아카이브에 잘 맞는 구독 기능입니다.
  - VoidNews 이식 형태: Weekly와 AB를 합친 Atom feed 또는 각각의 feed를 정적 생성합니다.
  - 난이도: **M**
  - 기대 효용: 플랫폼 알고리즘 없이 새 브리핑을 구독할 수 있습니다.

## 보류하거나 변형할 기능

- [ ] **편집 10점 평점은 그대로 이식하지 않기**
  - 무엇인지: 일부 항목에 출처명과 7/10·8/10 같은 평가가 붙습니다.
  - VoidNews 이식 형태: 현재 scoring의 근거 필드를 유지하고 공개 UI에는 검증 상태·화제성·발표 적합도를 서로 분리합니다.
  - 난이도: **M**
  - 기대 효용: 하나의 숫자가 공식성·중요도·취향을 뒤섞는 오해를 방지합니다.

- [ ] **매일 발행 cadence는 그대로 이식하지 않기**
  - 무엇인지: 날짜판이 매일 누적됩니다.
  - VoidNews 이식 형태: 내부 Radar는 매일 수집하되 공개판은 현재 격주 편집 cadence를 유지하고 긴급 업데이트만 예외로 둡니다.
  - 난이도: **M**
  - 기대 효용: 수집 커버리지는 높이면서 편집 품질과 운영 부담을 지킵니다.

- [ ] **커뮤니티 반응을 메인 뉴스와 동일 레벨에 두지 않기**
  - 무엇인지: HN·Reddit 반응이 독립 섹션을 이룹니다.
  - VoidNews 이식 형태: `community radar`에 보존하고 공식 1차 출처를 확보한 사건만 main/normal 카드로 승격합니다.
  - 난이도: **S**
  - 기대 효용: 화제성과 사실 검증을 분리합니다.

## 추천 묶음

### A. 최소 이식 — S 중심

- 날짜 고정 URL
- 호수+날짜
- 출처별 수집 건수
- 공식 링크 라벨
- HN/Reddit 원 점수 병기
- cadence 표시

**예상:** 작은 변경으로 Radar 투명성과 아카이브 탐색성이 바로 개선됩니다.

### B. 독자 행동형 — M 중심

- 섹션별 고정 질문
- `takeaway / tryNow / whyNow`
- Top/compact 밀도 차등
- 검색·태그 필터
- 공개 Markdown 복사

**예상:** 뉴스 목록을 발표·수업·실습으로 연결하는 경험이 강화됩니다.

### C. 출판 플랫폼형 — L 포함

- 내부 Deep Dive
- 월별 달력
- RSS/Atom
- 메신저 공유
- 검증 상태 라벨

**예상:** VoidNews를 단순 주간 목록에서 장기 검색·구독 가능한 지식 아카이브로 확장합니다.
